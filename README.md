Academia Digital – Landing Page Educativa

Proyecto desarrollado como práctica de maquetación y diseño web del curso Desarrollo Web Fullstack con React, Firebase y Javascript, recreando una landing page moderna para una plataforma de cursos online.  
Incluye secciones completas, navegación interna, páginas adicionales (Iniciar sesión) y una estructura organizada haciendo uso de HTML y CSS.

<!-- ============================
Descripción del Proyecto:
============================= -->
Este proyecto consiste en el desarrollo completo de una Landing Page Educativa, compuesta por varias secciones:

- Header  
- Banner 
- Sección de "Cursos"
- Sección de "Nosotros"
- Sección de "Instructores"  
- Sección de "Testimonios"   
- Página de inicio de sesión
El objetivo del proyecto es practicar la maquetación web, buenas prácticas, semántica HTML5, diseño responsivo e implementación de estilos con CSS.

<!-- ============================
Tecnologías Utilizadas:
============================= -->
- HTML5: Estructura del proyecto y semántica,
- CSS3: Estilos, layout, flexbox y diseño.
- Google Fonts: Tipografías personalizadas,
- GitHub: Control de versiones y despliegue
- Flexbox: Maquetación en todas las secciones
- Visual Studio Code: Editor de código
